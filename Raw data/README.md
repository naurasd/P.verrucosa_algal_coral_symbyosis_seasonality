This directory contains all raw data collected for the project.

[Environmental Raw Data](Environmental%20Raw%20Data.xlsx) contains all raw data measured for the following parameters:
- water temperature
- photosynthetically active radiation
- salinity
- dissolved oxygen
- dissolved nutrients (nitrite, nitrate, ammonium, phosphate)
- total dissolved carbon
- total dissolved nitrogen

In [Coral fragment info + surface area.xlsx](Coral%20fragment%20info%20%2B%20surface%20area.xlsx) you find general info about the collected coral fragments:
- sampling dates
- which initial samples fo the spring season have been pooled
- fragment weight
- the calculated fragment surface area
- the weight and volume of the tissue slurry generated from each fragment
- concentrations of DNA extracted from each slurry aliquot

[Algal symbiont data + nifH.xlsx](Algal%20symbiont%20data%20%2B%20nifH.xlsx) contains data on algal symbionts and diazotrophy:
- the total number of algal symbiont cells counted in each slurry aliquot
- the mitotic index of symbiont cells
- chlorophyll _a_ content per symbiont cell
- Raw data for 16S rRNA and _nifH_ gene copy numbers and fold change calculations for relative _nifH_ gene abundance

[Elemental and isotopic data.xlsx](Elemental%20and%20isotopic%20data.xlsx) contains data on elemental carbon and nitrogen composition and stable isotope ratios for coral host tissue and algal symbiont samples:
- sheets denoted as `non-acidified` contain data for elemental nitrogen and δ<sup>15</sup>N isotopic ratios
- sheets denoted as `acidified` contain data for elemental organic carbon and δ<sup>13</sup>C isotopic ratios
